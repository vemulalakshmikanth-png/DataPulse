# DataPulse
DataPulse is a production-grade, automated monitoring suite for Databricks data platforms, built in PySpark. It consists of two scheduled notebooks — one focused on performance optimization (small files, missing clustering, long-running/failing jobs, cost estimates) and one on metadata governance (schema change detection, naming standard violations, data freshness, and Unity Catalog lineage tracing). Each notebook runs nightly, compares against historical snapshots, and delivers a rich HTML email report to configured distribution lists. Designed with idempotency, gap-handling, 90-day retention, and zero manual intervention in mind.
